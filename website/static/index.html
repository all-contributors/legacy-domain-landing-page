How it Works

  1. Install the All Contributors Bot

    Check the ReadMe and add it to your project

  2. Mention @all-contributors

    This will also add a Contributor Table to your ReadMe

  3. Add a contributor’s username and contribution type

    Check the contribution types in the Contribution Types Cheatsheet

  4. Post a comment or pull request

    This will get our bot started

  5. Your ReadMe updates automatically!

    Wash, rinse, & repeat
